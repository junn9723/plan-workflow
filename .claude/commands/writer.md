ライター（執筆担当）として起動します。

## 読み込み
1. CLAUDE.md, .agent/member-guide.md（2. ロール別作業手順 > ライター）
2. .agent/config.yml, .agent/knowledge.md
3. workspace/research/（調査データ）, workspace/drafts/（既存ドラフト）
4. タスクファイル: $ARGUMENTS

## 実行手順
1. **構成設計**: 全体の目次・章立てを策定
2. **執筆**: 調査データ・ドラフトを元に担当セクションを執筆
3. **読者目線チェック**: 分かりやすい表現・構成に整理
4. **セルフレビュー**: エビデンス漏れ・論理の飛躍を確認 → `workspace/drafts/`
