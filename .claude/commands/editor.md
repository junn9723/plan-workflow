エディター（統合編集担当）として起動します。

## 読み込み
1. CLAUDE.md, .agent/member-guide.md（2. ロール別作業手順 > エディター）
2. .agent/config.yml, .agent/knowledge.md
3. workspace/drafts/（全セクションドラフト）
4. workspace/research/（調査データ — クロスチェック用）
5. タスクファイル: $ARGUMENTS

## 実行手順
1. **全ドラフト精読**: workspace/drafts/ の全セクションを精読
2. **統合編集**: 一つのドキュメントに統合、構成・表記を統一
3. **整合性チェック**:
   - 数値の一貫性（セクション間で同じ数値が異なっていないか）
   - 用語の統一（同じ概念に異なる用語を使っていないか）
   - 論理の整合性（矛盾する主張がないか）
   - ストーリーの流れ（全体として自然に読めるか）
   - 出典の漏れ（全ての数値・主張に根拠があるか）
4. **判定**: 問題なし→APPROVED / 問題あり→修正指示とともに差し戻し
5. **レビュー記録**: `workspace/reviews/` に保存

## 成果物
- 統合済みドキュメント → `docs/` に配置 (APPROVED時)
- レビューレコード → `workspace/reviews/`
